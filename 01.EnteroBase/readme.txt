01. Enterobase_7MLST.txt
Downloaded from the EnteroBase database. This file contains seven-gene multilocus sequence typing (MLST) profiles and sequence type assignments generated by EnteroBase based on the allelic profiles of the seven housekeeping marker genes.

02. Enterobase_HCcluster.txt
Downloaded from the EnteroBase database. This file contains core-genome MLST (cgMLST) clustering information, including HC1100 cluster assignments computed by EnteroBase. Note that the HC1100 cluster IDs in this file differ from the HC1100 cluster IDs newly generated and used in the present study.
