### `00.annotation.sh`
Script for annotating virulence factors (VFs), antimicrobial resistance genes (ARGs), and gene modules in newly assembled genomes.

### `01.VF&ARG.xlsx`
Functional annotations of virulence factors (VFs) and antimicrobial resistance genes (ARGs), provided in separate worksheets. The file includes:
- Identification and annotation of virulence genes
- Identification and annotation of antimicrobial resistance genes
- Gene descriptions and functional classifications

### `02.gene_modules.xlsx`
Detailed annotations of the 14 gene modules enriched in the dominant strains. The file includes:
- Gene composition of each module
- Nucleotide sequences
- Predicted gene and module functions
- Genomic context information, including flanking genes surrounding the focal modules

### `03.pangenomeGraph_light.gexf`
Pangenome network generated by PPanGGOLiN during gene module identification. In this network:
- Each node represents a gene family
- Edges represent genomic neighborhood relationships between gene families
- Gene module assignments are provided 
- The file is in GEXF format and can be visualized and explored using software such as Gephi.

### `04.enriched_functions.R`
R script used to identify gene modules enriched in the top 10 dominant strains. The same analytical framework was applied to virulence factors (VFs) and antimicrobial resistance genes (ARGs) to detect functions associated with strain dominance.
