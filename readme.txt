00. New_hc1100.txt
Generated in this study by integrating complete Escherichia coli genomes from NCBI, representative HC1100 genomes from EnteroBase, and the isolates analyzed in this work. The file contains the newly inferred HC1100 cluster assignments derived from the cgMLST-based clustering analysis.

00. code.sh
Shell script implementing the analysis pipeline used to generate the newly defined HC1100 clusters described in this study.
